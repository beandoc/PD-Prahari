# **App Name**: DialysisCare

## Core Features:

- Patient Overview: Dashboard providing an overview of patient's health metrics.
- Exchange Data Management: Add, edit, and view detailed PD exchange data for each patient.
- Vital Signs Monitoring: Record and track vital signs, including blood pressure, heart rate and weight.
- Lab Result Tracking: Enter, review, and visualize clinical lab results, and spot any anomalies over time.
- Medication Management: Maintain and visualize patient's medication list, dosages and frequencies.
- AI Medication Tool: AI-powered tool that monitors patient data and suggests potential medication adjustments. It should use reasoning, guided by the collected clinical history, to include suggestions when relevant.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) for a calming and trustworthy feel, promoting a sense of health and well-being.
- Background color: Very light blue (#F0F8FF), almost white, to ensure clarity and readability.
- Accent color: Soft teal (#70D1D1) to highlight important actions and key data points.
- Font: 'Inter', a sans-serif font known for its modern, clean design and excellent legibility.
- Use a set of consistent and professional icons that are easy to understand, following a minimalist style.
- Maintain a clean and structured layout for easy navigation. Use white space effectively to avoid overwhelming users with too much data.
- Subtle transitions and animations to enhance user experience, like smooth data updates and form interactions.
